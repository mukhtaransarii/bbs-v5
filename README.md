## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/mukhtaransarii/bbs-v5.git
cd bbs-v5

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

```bash
# Build for production
npm run build
```
